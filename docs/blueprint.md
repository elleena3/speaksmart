# **App Name**: SpeakSmart Evaluator

## Core Features:

- Virtual Assessment Environment: Conduct speaking practice and performance assessments directly within the app.
- AI-Powered Feedback: Provide automated evaluation feedback to students by leveraging voice recognition and analysis. LLM will use information from the speaking activity prompt and expected format as a tool, along with student's voice recording, to generate feedback.
- Personalized Review: Enable students to review the AI feedback to promote self-reflection.
- Student Feedback to Teacher: Provide a feature for students to input feedback to teachers about the assessment activities.
- Content Improvement Tools: Offer tools for teachers to refine the evaluation content based on the data coming from the application.
- Performance Dashboards: Display overall class performance and evaluation metric distributions.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to promote trust and clear communication.
- Background color: Light gray (#F5F5F5), which keeps the interface clean and prevents distraction from content.
- Accent color: Yellow (#FFD700) to highlight actionable items such as buttons or important feedback elements.
- Body and headline font: 'PT Sans' (sans-serif) for its modern and readable design suitable for educational content.
- Utilize clean and clear icons that represent assessment elements.
- Maintain a structured layout that guides teachers and students effectively.
- Use micro-interactions to indicate evaluation activities.
